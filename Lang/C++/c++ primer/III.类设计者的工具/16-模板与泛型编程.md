# 16.1.1 函数模板
- 一个函数就像是一个公式
```c++
template <typename T>
int compare(const T& a, const T& b) {
  if(a < b) return -1;
  if(b < a) return 1;
  return 0;
}
```
- 以关键字`template`开始
- 跟一个`模板参数列表`
  - 模板参数列表不可为空

- 编写泛型代码的两个原则
  - 模板中的函数参数是const的引用
  - 函数体中的条件判断仅使用`<`比较运算

## 16.1.2 类模板
- 一个类模板的每个实例都形成一个独立的类
- 对一个实例化了的类模板,其成员只有在使用时才被实例化
