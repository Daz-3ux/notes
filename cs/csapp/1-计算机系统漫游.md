# 1.2 程序被其他程序翻译成不同格式
- `编译系统`:**预处理器 编译器 汇编器 链接器**
```
hello.c文件
经过 预处理器(cpp)
hello.i
经过 编译器(cgi)
hello.s
经过 汇编器(as)
hello.o
经过 链接器(ld)
hello:可执行目标程序
```

# 1.5 高速缓存至关重要
- `高速缓存存储器`:针对处理器和主存之间的差异,系统设计者采用的更小更快的存储设备 **cache memory**
- 就是位于CPU的L1cache,L2chahe..
- 意识到高速缓存存储器的存在并使用它可以将程序的性能提高一个数量级
