# 编译器和会处理C++的空类

```
class Empty { };
```

- 会自动生成public且inline的一系列默认函数
- 当这些函数被需要时会被编译器创造
```c++
class Empty { 
public:
  Empty() { }                               // deafult构造函数
  Empty(cosnt Empty &rhs) { }               // copy构造函数
  ~Empty() { }                              // 析构函数

  Empty operator=(const Empty &rhs) { }     // copy assignment操作符
};
```
